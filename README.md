# @hexlet/code - Slack-like Chat Application

A real-time chat application inspired by Slack, built with React and Vite.

## Features

- Real-time messaging
- Channel management
- User authentication
- Responsive design

## Demo

Live demo: [https://your-app-name.onrender.com](https://your-app-name.onrender.com)

## Technologies

- React 18
- Vite
- @hexlet/chat-server
- JavaScript (ES Modules)
- ESLint 8.57.1

## Setup

```bash

make install
make build
make start
```

## Development
```bash

cd frontend && npm run dev
```

## Deployment
The application is deployed on Render. See render.yaml for configuration.
